# 🏆 TaskMaster - Eder News Edition

![Palmeiras Theme](https://img.shields.io/badge/Theme-Palmeiras-006437?style=for-the-badge)
![Supabase](https://img.shields.io/badge/Supabase-Backend-3ECF8E?style=for-the-badge&logo=supabase)

Um sistema de gerenciamento de tarefas moderno, performático e elegante, desenvolvido com foco em experiência do usuário e persistência em nuvem.

## 🚀 Funcionalidades

- **CRUD Completo**: Adicione, liste, conclua e exclua tarefas em tempo real.
- **Categorização**: Organize-se com categorias (Pessoal, Trabalho, Estudos).
- **Prioridades**: Identifique o que é urgente com tags de prioridade.
- **Dashboards**: Visualize seu desempenho com gráficos dinâmicos de produtividade.
- **Calendário**: Acompanhe seus prazos em uma visão mensal integrada.
- **Persistência Supabase**: Seus dados seguros e acessíveis de qualquer lugar.

## 🛠️ Tecnologias

- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+).
- **Backend-as-a-Service**: [Supabase](https://supabase.com/).
- **Ícones**: [Lucide Icons](https://lucide.dev/).
- **Design**: Glassmorphism & Palmeiras Palette.

## 📦 Como Instalar

1. Clone o repositório:
   ```bash
   git clone https://github.com/edernews/sistemas_tarefas.git
   ```

2. Configure o Supabase:
   - Renomeie o arquivo `config.example.js` para `config.js`.
   - Insira sua `SUPABASE_URL` e `SUPABASE_KEY` no arquivo.

3. Execute:
   - Basta abrir o `index.html` em qualquer navegador moderno.

## 🔒 Segurança

As chaves de API utilizadas no frontend são do tipo `publishable`/`anon`, seguras para exposição pública. No entanto, o arquivo `config.js` está no `.gitignore` para manter o ambiente local isolado.
